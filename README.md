## API Endpoints

GET /sos → Retrieve SOS requests

- AI-based disaster severity prediction

## System Workflow

1. User enters disaster data
2. AI predicts severity
3. User sends SOS request
4. Backend stores data
5. Admin dashboard displays SOS alerts
